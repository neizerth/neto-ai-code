## Демо для презентации (07)

Здесь — рабочие примеры для скринкаста:

- `02/streamlit_app.py` — интерфейс на Streamlit
- `03/gradio_app.py` — интерфейс на Gradio
- `01/llm.py` — общий обработчик (вызов модели + проверки + ошибки)

Сценарий: **ответ студенту в учебном чате** (вопрос + тон ответа).

### Что нужно заранее

- Python 3.10+
- Ключ OpenAI в переменной окружения `OPENAI_API_KEY`

### Быстрый запуск

1) Создайте и активируйте окружение:

```bash
python -m venv .venv
source .venv/bin/activate
```

2) Поставьте зависимости:

```bash
pip install -r 01/requirements.txt
```

3) Добавьте ключ:

```bash
cp 01/.env.example 01/.env
```

Откройте `01/.env` и вставьте `OPENAI_API_KEY=...`.

4) Запуск Streamlit:

```bash
streamlit run 02/streamlit_app.py
```

5) Запуск Gradio:

```bash
python 03/gradio_app.py
```

### Как проверить ошибки (для демо)

- **пустой ввод**: нажмите кнопку с пустым полем
- **слишком длинный ввод**: вставьте очень длинный текст (порог в коде)
- **ошибка доступа**: временно очистите `OPENAI_API_KEY` и запустите снова
