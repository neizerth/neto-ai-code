## Исправь ошибку в скрипте Димы
На экране Димы открыт скрипт обработки заказов. Программа падает с ошибкой `KeyError: 'order_id'`, потому что не у всех заказов есть поле `order_id`. Исправьте одну строку так, чтобы скрипт не завершался с ошибкой. Используйте `get()` и подставьте строку `"без номера"`, если ключ отсутствует

### Начальный код задания
```python
orders = [
    {"order_id": "A-1001", "item": "Наушники"},
    {"item": "Клавиатура"},
    {"order_id": "A-1003", "item": "Мышь"},
]

for order in orders:
    order_id = order["order_id"]  # исправьте эту строку
    print(f"Заказ {order_id} отправлен в обработку")

print("Обработка завершена")
```

### Эталонное решение эксперта
```python
orders = [
    {"order_id": "A-1001", "item": "Наушники"},
    {"item": "Клавиатура"},
    {"order_id": "A-1003", "item": "Мышь"},
]

for order in orders:
    order_id = order.get("order_id", "без номера")
    print(f"Заказ {order_id} отправлен в обработку")

print("Обработка завершена")

```
### Ожидаемый ответ
```python
[
    "Заказ A-1001 отправлен в обработку",
    "Заказ без номера отправлен в обработку",
    "Заказ A-1003 отправлен в обработку",
    "Обработка завершена"
]
```

### Подсказка:
Если ключа в словаре может не быть, используйте `get()`. У этого метода можно сразу указать запасное значение, которое вернётся вместо ошибки.

### Тест:

```
import subprocess
from pathlib import Path


def test_solution():
    code = Path("main.py").read_text(encoding="utf-8")

    assert (
        '.get("order_id", "без номера")' in code
        or ".get('order_id', 'без номера')" in code
    ), "Используйте get() и подставьте значение 'без номера'."

    result = subprocess.run(
        ["python", "main.py"],
        capture_output=True,
        text=True,
        encoding="utf-8"
    )

    assert result.returncode == 0, "Программа не должна завершаться с ошибкой."

    output = result.stdout.strip().splitlines()

    expected = [
        "Заказ A-1001 отправлен в обработку",
        "Заказ без номера отправлен в обработку",
        "Заказ A-1003 отправлен в обработку",
        "Обработка завершена"
    ]

    assert output == expected, "Проверьте вывод программы."
```
